Noker Logos – Experiment Project is the research and engineering subsystem of the main [Noker](https://github.com/kiritough/Noker) project.

This is where the first implementations of future Noker features are created. Solutions that prove their value in Logos eventually become part of the main Noker project.

---

## Why a Separate Repository

The main Noker repository is for stable, verified solutions only.

Logos exists so that ideas can be explored, tested, and refined without affecting the stability of the main project. It is the place where things can be rewritten, broken, and rebuilt — as many times as needed — before they are mature enough to become part of Noker.

---

## Connection to Noker

Noker Logos – Experiment Project feeds into the main Noker project.

Research and builds happen here first. Solutions that prove their value move to [kiritough/Noker](https://github.com/kiritough/Noker).

**Main project → [kiritough/Noker](https://github.com/kiritough/Noker)**
